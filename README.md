# <img width="48" height="48" alt="image" src="https://github.com/user-attachments/assets/f5a35ffb-340d-41cf-9c49-db580a92ca96" /> Human Spleen SnRNA-seq and Spatial Transcriptomics Analysis
This repository documents the main workflows and scripts used in the paper "_Comprehensive spatial profiling of the spleen in murine and human myelofibrosis identifies targetable immune-stromal interactions_" from Dugué et al.
In particular, it documents all code used for the analysis of single-nuclei RNA sequencing (snRNA-seq) and Spatial Transcriptomics (Visium) data from the human MPN spleen, as well as the analysis of Visium HD data from the MF mouse models.
